# Quizable

Quizable is an interactive web-based quiz platform designed to transform boring study notes into engaging, bite-sized challenges. It empowers learners to master subjects through fun quizzes while providing instructors with tools to create and manage educational content.

## Features

### For Students
- **Take Quizzes**: Access a variety of quizzes across different subjects (C Programming, Java, HTML)
- **Interactive Interface**: Multiple-choice questions with real-time feedback
- **Score Tracking**: View scores and progress
- **Announcements**: Stay updated with important notices from instructors/admins

### For Instructors
- **Create Quizzes**: Build custom quizzes with multiple-choice questions
- **Manage Content**: Edit and delete existing quizzes
- **Analytics**: View student performance and quiz statistics
- **Results Dashboard**: Track student attempts and average scores

### For Admins
- **User Management**: View, block, verify, and delete user accounts
- **Quiz Moderation**: Oversee all quizzes and remove inappropriate content
- **Analytics Dashboard**: Monitor platform growth, top performers, and system stats
- **Announcements**: Post system-wide notices

### General Features
- **Role-Based Access**: Secure authentication with Student, Instructor, and Admin roles
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Custom Cursor**: Enhanced user experience with animated cursor effects
- **Local Storage**: Data persistence without requiring a backend server
- **Accessibility**: Built with accessibility best practices in mind

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Modern styling with CSS variables, gradients, and animations
- **Vanilla JavaScript**: DOM manipulation, event handling, and application logic
- **Local Storage API**: Client-side data persistence

## Installation & Running

1. **Clone or Download**: Download the project files to your local machine
2. **Open in Browser**: Simply open `index.html` in any modern web browser
3. **No Dependencies**: The application runs entirely in the browser with no external dependencies

## Usage

### Getting Started
1. Open `index.html` in your web browser
2. Click "Start Quizzing Free" or "Log in"
3. Choose your role (Student, Instructor, or Admin) and sign up or log in

### Creating an Account
- Select your role during signup
- Password must be at least 8 characters with a mix of letters and numbers/symbols
- Instructors require admin verification before full access

### Taking a Quiz
- Browse available quizzes by subject
- Click "Start Quiz" to begin
- Answer questions and see immediate feedback
- View final score upon completion

### Creating a Quiz (Instructors)
- Log in as an Instructor
- Click "+ New Quiz" in your dashboard
- Add quiz title, select subject, and create questions
- Each question supports 4 multiple-choice options

### Admin Functions
- Access comprehensive user and quiz management
- Post announcements visible to all users
- Monitor platform analytics and growth
